# 호텔 콜센터 적용 범위

현재 제품은 전화망에 음성을 직접 주입하는 장치가 아니다. 고객 쪽 수신 음성을 마이크 또는
Windows loopback으로 받아 일본어 텍스트로 보여주고, 직원은 일본어 답변을 입력한 뒤 번역문과
읽기 표기를 보고 직접 말한다.

```text
전화/소프트폰 수신음 → RemotePlus 입력 → 고객 원문 + 일본어 번역
직원 일본어 키보드 → 고객 언어 번역 + 가타카나/로마자 → 직원이 직접 발화
```

TAKACOM SA-600 같은 물리 전화기에서 송수신 채널을 소프트웨어가 직접 분리하려면 별도 오디오
인터페이스나 PBX/소프트폰 API가 필요하다. 이 브랜치는 그 하드웨어 의존성을 제거한 텍스트 보조
운영을 목표로 한다.

중요 예약 변경, 결제, 알레르기, 응급 상황, 법적 고지는 직원이 원문을 재확인하고 필요하면 전문
통역으로 전환한다. 현장 음성은 권리와 개인정보를 정리해 익명화한 뒤 회귀 테스트에 사용한다.
